# vi_memo_to_self

### vi (vim) 備忘録

自分用メモ

---

### 終了 / 保存

```
:q!
# 強制終了(Escを事前連打)

:q
# そのまま終了

:wq
# 上書き保存＋終了

:w
# 上書き保存
```

---

### 移動

コマンドモード

```
h　j　k　l
# 左・下・上・右 にそれぞれ移動

:number_hoge
# 行移動  (e.g. ":5" など)
```

---

### 削除

```
x
# カーソル位置の文字削除

dd
# 行を削除
```

---

### その他メモ

#### vi (ヴイアイ)

- Windowsでいうメモ帳の __Linux版__ (みたいなもの)


    ```
    # vi / vim での入り方

    $ vi hoge.txt

    $ vim hoge.txt
    ```

#### コマンドモード / 文字入力モード

おもに2つのモードがある。

- コマンドモード

  - 終了・保存・行削除 など

- 文字入力モード

  文字挿入・書き込み

  1. 起動  (直後はコマンドモードになってる)
  1. __文字入力モード__ へ切り替え
  1. __コマンドモード__ へ切り替え
  1. 終了
